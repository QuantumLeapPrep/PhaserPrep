Задачи по термодинамике (ОГЭ - повышенная сложность)

Задача 1

В калориметр, содержащий 200 г воды при температуре 20°C, поместили кусок льда массой 50 г при температуре -10°C. Какая температура установится в калориметре после установления теплового равновесия? Удельная теплоемкость воды 4200 Дж/(кг·К), удельная теплоемкость льда 2100 Дж/(кг·К), удельная теплота плавления льда 3,3⋅105 Дж/кг.

Решение:

Сначала лед нагревается до 0°C: Q1 = mл * cл * ΔT = 0.05 кг * 2100 Дж/(кг·К) * 10 К = 1050 Дж.

Затем лед плавится: Q2 = mл * λ = 0.05 кг * 3.3⋅105 Дж/кг = 16500 Дж.

Теперь посмотрим, сколько теплоты может отдать вода, охладившись до 0°C: Q3 = mв * cв * ΔT = 0.2 кг * 4200 Дж/(кг·К) * 20 К = 16800 Дж.

Так как Q3 > Q1 + Q2, то лед полностью растает. Найдем установившуюся температуру. Теплота, отданная водой: Q4 = mв * cв * (20 - T), а теплота, полученная растаявшим льдом для нагрева: Q5 = mл * cв * (T - 0).

Составляем уравнение теплового баланса: Q4 = Q1 + Q2 + Q5, или mв * cв * (20 - T) = 1050 + 16500 + mл * cв * T.

0.2 * 4200 * (20 - T) = 1050 + 16500 + 0.05 * 4200 * T.

16800 - 840T = 17550 + 210T.

1050T = -750.

T ≈ -0.71 °C.

Поскольку температура не может быть ниже 0°C (иначе бы замерзла вода), то в системе будет тепловое равновесие при 0°C, и часть воды замерзнет.

Ответ: 0°C.

Задача 2

Сколько дров нужно сжечь, чтобы обратить в пар 2 кг воды, взятой при температуре 20°C, если КПД нагревателя 25%? Удельная теплоемкость воды 4200 Дж/(кг·К), удельная теплота парообразования воды 2,3⋅106 Дж/кг, удельная теплота сгорания дров 107 Дж/кг.

Решение:

Сначала нагреваем воду от 20°C до 100°C: Q1 = mв * cв * ΔT = 2 кг * 4200 Дж/(кг·К) * 80 К = 672000 Дж.

Затем обращаем воду в пар при 100°C: Q2 = mв * L = 2 кг * 2.3⋅106 Дж/кг = 4600000 Дж.

Общее количество теплоты, необходимое для процесса: Q = Q1 + Q2 = 672000 + 4600000 = 5272000 Дж.

Так как КПД нагревателя 25%, то сжигаем дрова: η = Q / Qсгор, где Qсгор - количество теплоты, выделившееся при сгорании дров.

Qсгор = Q / η = 5272000 Дж / 0.25 = 21088000 Дж.

Теперь найдем массу дров: Qсгор = mд * q, где q - удельная теплота сгорания дров.

mд = Qсгор / q = 21088000 Дж / 107 Дж/кг = 2.1088 кг.

Ответ: Нужно сжечь примерно 2.11 кг дров.

Задача 3

В алюминиевом чайнике массой 0.5 кг нужно вскипятить 2 л воды, начальная температура которой 20°C. Какое количество теплоты потребуется для этого? Удельная теплоемкость алюминия 920 Дж/(кг·К), удельная теплоемкость воды 4200 Дж/(кг·К).

Решение:

Сначала нагреваем чайник: Q1 = mч * cч * ΔT = 0.5 кг * 920 Дж/(кг·К) * 80 К = 36800 Дж.

Затем нагреваем воду: Q2 = mв * cв * ΔT = 2 кг * 4200 Дж/(кг·К) * 80 К = 672000 Дж.

Общее количество теплоты: Q = Q1 + Q2 = 36800 + 672000 = 708800 Дж = 708.8 кДж.

Ответ: Потребуется 708.8 кДж теплоты.

Задача 4

В железный сосуд массой 100 г налили 200 г воды при температуре 15°C. После того как в сосуд опустили кусок меди массой 50 г, нагретый до 100°C, температура воды повысилась до 18°C. Определите удельную теплоемкость меди. Удельная теплоемкость воды 4200 Дж/(кг·К), удельная теплоемкость железа 460 Дж/(кг·К).

Решение:

Теплота, отданная медью: Q1 = mм * cм * (100 - 18) = 0.05 * cм * 82 = 4.1cм.

Теплота, полученная водой: Q2 = mв * cв * ΔT = 0.2 кг * 4200 Дж/(кг·К) * 3 К = 2520 Дж.

Теплота, полученная железом: Q3 = mж * cж * ΔT = 0.1 кг * 460 Дж/(кг·К) * 3 К = 138 Дж.

Уравнение теплового баланса: Q1 = Q2 + Q3, или 4.1cм = 2520 + 138 = 2658.

cм = 2658 / 4.1 ≈ 648.3 Дж/(кг·К).

Ответ: Удельная теплоемкость меди примерно 648.3 Дж/(кг·К).

Задача 5

В воду массой 1 кг при температуре 20°С бросили кусок стали массой 200 г, нагретый до 400°С. Температура воды поднялась до 24°С. Определите удельную теплоемкость стали. Удельная теплоемкость воды 4200 Дж/(кг·К).

Решение:

Теплота, отданная сталью: Q1 = mст * cст * (400 - 24) = 0.2 * cст * 376 = 75.2cст.

Теплота, полученная водой: Q2 = mв * cв * ΔT = 1 кг * 4200 Дж/(кг·К) * 4 К = 16800 Дж.

Уравнение теплового баланса: Q1 = Q2, или 75.2cст = 16800.

cст = 16800 / 75.2 ≈ 223.4 Дж/(кг·К).

Ответ: Удельная теплоемкость стали примерно 223.4 Дж/(кг·К).
