Задачи на расчет количества теплоты

Задача 1

Какое количество теплоты необходимо для нагревания 2 кг воды от 20°C до 100°C? (Удельная теплоемкость воды c = 4200 Дж/(кг·°C))

  • Дано:
    • m = 2 кг
    • T₁ = 20 °C
    • T₂ = 100 °C
    • c = 4200 Дж/(кг·°C)
  • Найти: Q
  • Решение:
    • Формула для количества теплоты: Q = cmΔT
    • Вычислим: Q = 4200 * 2 * (100 - 20) = 672000 Дж
  • Ответ: 672000 Дж (672 кДж)

Задача 11

На сколько градусов нагреется кусок меди массой 400 г, если ему передать 10,8 кДж теплоты? (Удельная теплоемкость меди c = 380 Дж/(кг·°C))

  • Дано:
    • m = 400 г = 0,4 кг
    • Q = 10,8 кДж = 10800 Дж
    • c = 380 Дж/(кг·°C)
  • Найти: ΔT
  • Решение:
    • Формула для количества теплоты: Q = cmΔT
    • Вычислим: ΔT = Q/(cm) = 10800 / (380 * 0.4) = 71,05 °C
  • Ответ: 71,05 °C

Задача 12

Рассчитайте количество теплоты, которое необходимо для плавления куска свинца массой 5 кг, взятого при температуре 27°C? Температура плавления свинца равна 327°C, удельная теплоёмкость свинца 140 Дж/(кг*К), удельная теплота плавления 0,25 * 10^5 Дж/кг.

  • Дано:
    • m = 5 кг
    • T₁ = 27 °C
    • T_пл = 327 °C
    • c = 140 Дж/(кг·°C)
    • λ = 0,25 * 10^5 Дж/кг
  • Найти: Q
  • Решение:
    • Формула для количества теплоты: Q = cmΔT + λm
    • Вычислим: Q = 140 * 5 * (327 - 27) + 0.25 * 10^5 * 5 = 105000 + 1250000 Дж
  • Ответ: 1355000 Дж (1,355 МДж)

Задача 14

Воде массой 5 кг, находящейся при температуре 20 °С, передано количество теплоты, равное 1680 кДж. Определите конечную температуру воды, если её удельная теплоёмкость равна 4200 Дж/(кг °С).

  • Дано:
    • m = 5 кг
    • T₁ = 20 °C
    • Q = 1680 кДж = 1680000 Дж
    • c = 4200 Дж/(кг·°C)
  • Найти: T₂
  • Решение:
    • Формула для количества теплоты: Q = cmΔT
    • Выразим конечную температуру: T₂ = T₁ + Q/(cm)
    • Вычислим: T₂ = 20 + 1680000/(4200 * 5) = 100 °C
  • Ответ: 100 °C

Задача 15

Кусок льда массой 2 кг при температуре 0 °С помещают в сосуд с водой массой 10 кг, имеющей температуру 25 °С. Какая установится общая температура в сосуде? Удельная теплоёмкость воды равна 4200 Дж/(кг °С), а удельная теплота плавления льда равна 3,3 * 10^5 Дж/кг.

  • Дано:
    • m_л = 2 кг
    • m_в = 10 кг
    • T_в = 25 °C
    • c = 4200 Дж/(кг·°C)
    • λ = 3,3 * 10^5 Дж/кг
  • Найти: T
  • Решение:
    • m_в * c * (25 - T) = λ * m_л + m_л * c * (T - 0)
    • 10 * 4200 * (25 - T) = 3,3 * 10^5 * 2 + 2 * 4200 * (T - 0)
    • Решаем уравнение: T ≈ 9,29 °C
  • Ответ: 9,29 °C

Задача 16

В алюминиевом чайнике массой 0,5 кг нагрели воду объёмом 2 л от 20 °С до кипения. Сколько на это было затрачено энергии? Удельная теплоёмкость воды равна 4200 Дж/(кг °С), удельная теплоёмкость алюминия 920 Дж/(кг °С). Плотность воды составляет 1000 кг/м3.

  • Дано:
    • m_ал = 0,5 кг
    • V_в = 2 л = 0,002 м³
    • T₁ = 20 °C
    • T₂ = 100 °C
    • c_в = 4200 Дж/(кг·°C)
    • c_ал = 920 Дж/(кг·°C)
    • ρ_в = 1000 кг/м³
  • Найти: Q
  • Решение:
    • Q = m_в * c_в * ΔT + m_ал * c_ал * ΔT
    • Q = (ρ_в * V_в) * c_в * ΔT + m_ал * c_ал * ΔT
    • Q = (1000 * 0,002) * 4200 * (100 - 20) + 0,5 * 920 * (100 - 20)
    • Q = 672000 + 36800 = 708800 Дж
  • Ответ: 708800 Дж (708,8 кДж)

Задача 17

На спиртовке нагрели 200 г воды от 20°С до 70 °С. Сколько спирта сгорело при этом, если КПД спиртовки 40%? Удельная теплоемкость воды с = 4200 Дж/(кг °С), удельная теплота сгорания спирта q = 2,7 * 10^7 Дж/кг.

  • Дано:
    • m_в = 200 г = 0,2 кг
    • T₁ = 20 °C
    • T₂ = 70 °C
    • η = 40% = 0,4
    • c = 4200 Дж/(кг·°C)
    • q = 2,7 * 10^7 Дж/кг
  • Найти: m_сп
  • Решение:
    • η * q * m_сп = c * m_в * ΔT
    • m_сп = (c * m_в * ΔT) / (η * q)
    • m_сп = (4200 * 0.2 * (70 - 20)) / (0.4 * 2,7 * 10^7) = 0.00389 кг
  • Ответ: 0.00389 кг (3.89 г)

Задача 18

В сосуд с водой при температуре 20° С поместили кусок льда, масса которого равна 1 кг, и стали нагревать. Какое количество теплоты потребуется для того, чтобы лед растаял и вода нагрелась до 50° С? Удельная теплоемкость воды равна 4200 Дж/(кг °С), а удельная теплота плавления льда равна 3,3 * 10^5 Дж/кг.

  • Дано:
    • m = 1 кг
    • T₁ = 20 °C
    • T₂ = 50 °C
    • c = 4200 Дж/(кг·°C)
    • λ = 3,3 * 10^5 Дж/кг
  • Найти: Q
  • Решение:
    • Теплота на плавление льда: Q₁ = λ * m
    • Теплота на нагрев воды: Q₂ = c * m * (50 - 0)
    • Общее количество теплоты: Q = Q₁ + Q₂
    • Вычислим: Q₁ = (3,3 * 10^5 * 1) = 330000 Дж
    • Вычислим: Q₂ = (4200 * 1 * (50 - 0)) = 210000 Дж
    • Вычислим: Q = Q₁ + Q₂ = 330000 + 210000 = 540000 Дж
  • Ответ: 540000 Дж (540 кДж)
