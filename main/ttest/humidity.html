Задачи на влажность воздуха

Задача 1

В комнате объемом 50 м³ содержится 600 г водяного пара при температуре 20°C. Определите абсолютную влажность воздуха.

  • Дано:
    • V = 50 м³
    • m = 600 г = 0,6 кг
  • Найти: ρ
  • Решение:
    • Формула для абсолютной влажности: ρ = m/V
    • Вычислим: ρ = 0,6 кг / 50 м³ = 0,012 кг/м³
  • Ответ: 0,012 кг/м³ (12 г/м³)

Задача 2

Относительная влажность воздуха в комнате при 25°C составляет 70%. Определите абсолютную влажность, если давление насыщенного пара при этой температуре составляет 23,8 мм рт. ст.

  • Дано:
    • φ = 70%
    • Pнас = 23,8 мм рт. ст.
  • Найти: P
  • Решение:
    • Формула для относительной влажности: φ = (P / Pнас) * 100%
    • Выразим P: P = (φ / 100) * Pнас
    • Вычислим: P = (70 / 100) * 23,8 = 16,66 мм рт. ст.
  • Ответ: 16,66 мм рт. ст.

Задача 3

При температуре 30°C относительная влажность воздуха равна 80%. Давление насыщенного водяного пара при этой температуре составляет 31,8 мм рт. ст., а плотность насыщенного пара 30.04 г/м³. Определите абсолютную влажность воздуха в кг/м³.

  • Дано:
    • φ = 80%
    • Pнас = 31,8 мм рт. ст.
    • ρнас = 30.04 г/м³ = 0.03004 кг/м³
  • Найти: ρ
  • Решение:
    • Формула для относительной влажности: φ = (ρ / ρнас) * 100%
    • Выразим ρ: ρ = (φ / 100) * ρнас
    • Вычислим: ρ = (80 / 100) * 0.03004 кг/м³ = 0.024032 кг/м³
  • Ответ: 0.024032 кг/м³

Задача 4

При температуре 25°C давление водяного пара в воздухе составляет 1.7 кПа, а давление насыщенного водяного пара при этой температуре равно 3.2 кПа. Определите относительную влажность воздуха.

  • Дано:
    • p = 1.7 кПа
    • pнас = 3.2 кПа
  • Найти: φ
  • Решение:
    • Формула для относительной влажности: φ = (p / pнас) * 100%
    • Вычислим: φ = (1.7 / 3.2) * 100% = 53.125%
  • Ответ: 53.125%

Задача 5

В 5 м³ воздуха при температуре 293 К находится 87 г водяного пара. Найти относительную влажность.

  • Дано:
    • V = 5 м³
    • m = 87 г = 0,087 кг
    • T = 293 К
  • Найти: φ
  • Решение:
    • p = (m/V) * (R * T / μ)
    • p0(20°C) = 2330 Па
    • φ = (p / p0) * 100%
  • Ответ: φ

Задача 6

Давление водяного пара в воздухе при 25°C составляет 1,4 кПа. Определите относительную влажность воздуха, если давление насыщенного водяного пара при этой температуре равно 3,2 кПа.

  • Дано:
    • p = 1.4 кПа
    • p_0 = 3.2 кПа
  • Найти: φ
  • Решение:
    • φ = (p / p_0) * 100%
    • φ = (1.4 / 3.2) * 100% = 43,75%
  • Ответ: 43,75%

Задача 7

Относительная влажность в комнате при 20°C равна 60%. Какова абсолютная влажность, если давление насыщенного водяного пара при этой температуре равно 2,33 кПа?

  • Дано:
    • φ = 60%
    • p_0 = 2.33 кПа
    • T = 20°C
  • Найти: ρ
  • Решение:
    • p = φ * p_0 = 0.6 * 2.33 ≈ 1,4 кПа
    • ρ = (p * μ) / (R * T)
    • μ = 0,018 кг/моль
    • R = 8,31 Дж/(К * моль)
    • T = 293 К
    • ρ = 0.01 кг/м³ = 10 г/м³
  • Ответ: 10 г/м³

Задача 8

В закрытом сосуде объемом 2 м3 находится воздух с относительной влажностью 70% при температуре 20°C. Определить массу водяного пара в воздухе.

  • Дано:
    • V = 2 м³
    • φ = 70% = 0.7
    • T = 20°C = 293 K
  • Найти: m
  • Решение:
    • P_0 = 2,33 кПа
    • P = φ * P_0 = 0.7 * 2,33 = 1.63 кПа
    • m = (PVμ) / (RT) = (1631.3 * 2 * 0.018) / (8.31 * 293) = 0,024 кг
  • Ответ: 0.024 кг

Задача 9

В комнате при температуре 20 °С психрометр показывает разность температур 5 °С. Определите относительную влажность воздуха. (Предположим, что по психрометрической таблице при 20°C и разности 5°C относительная влажность составляет 65%).

  • Дано:
    • T = 20 °C
    • ΔT = 5 °C
    • φ (из таблицы) = 65%
  • Найти: φ
  • Решение:
    • По психрометрической таблице, при температуре 20°C и разности температур 5°C, относительная влажность составляет 65%.
  • Ответ: 65%

Задача 10

Найдите абсолютную и относительную влажность воздуха при температуре 29 °С, если точка росы 17 °С. Давление насыщенного пара при 17 °С составляет 1.94 кПа, а при 29 °С - 4.01 кПа.

  • Дано:
    • t = 29 °С
    • tр = 17 °С
    • Pнас, 17 = 1.94 кПа
    • Pнас, 29 = 4.01 кПа
  • Найти: ρ, φ
  • Решение:
    • Абсолютная влажность: Абсолютная влажность равна парциальному давлению водяного пара, которое равно давлению насыщенного пара при точке росы: \( p = P_{нас, 17} = 1.94 кПа \)
    • Абсолютная влажность: Воспользуемся уравнением связи парциального давления и плотности: \( \rho = \frac{p \cdot \mu}{R \cdot T} \)
    • Относительная влажность: φ = (Pнас, 17 / Pнас, 29) * 100%
    • Вычислим: φ = (1.94 кПа / 4.01 кПа) * 100% = 48.38%
  • Ответ: φ = 48.38%
