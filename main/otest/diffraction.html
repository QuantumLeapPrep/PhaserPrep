Задачи на Дифракцию Света

Решите следующие задачи, чтобы закрепить свои знания о дифракции света.

Задача 1

На щель шириной 0,1 мм падает нормально монохроматический свет с длиной волны 500 нм. Определите угол, под которым наблюдается первый дифракционный минимум.

  • Дано:
    • a = 0,1 мм = 0,1 * 10⁻³ м (ширина щели)
    • λ = 500 нм = 500 * 10⁻⁹ м (длина волны)
    • m = 1 (первый минимум)
  • Найти: θ (угол дифракции)
  • Решение:
    • Условие минимума: a * sin(θ) = m * λ
    • Выразим sin(θ): sin(θ) = (m * λ) / a
    • Вычислим: sin(θ) = (1 * 500 * 10⁻⁹ м) / (0,1 * 10⁻³ м) = 0,005
    • Найдем θ: θ = arcsin(0.005) ≈ 0,29°
  • Ответ: ≈ 0,29°

Задача 2

Дифракционная решетка содержит 100 штрихов на 1 мм. На решетку нормально падает свет с длиной волны 600 нм. Определите угол, под которым наблюдается максимум второго порядка.

  • Дано:
    • N = 100 штрихов/мм = 10⁵ штрихов/м (плотность штрихов)
    • λ = 600 нм = 600 * 10⁻⁹ м (длина волны)
    • m = 2 (порядок максимума)
  • Найти: θ (угол дифракции)
  • Решение:
    • Период решетки: d = 1 / N = 1 / 10⁵ м = 10⁻⁵ м
    • Условие максимума: d * sin(θ) = m * λ
    • Выразим sin(θ): sin(θ) = (m * λ) / d
    • Вычислим: sin(θ) = (2 * 600 * 10⁻⁹ м) / 10⁻⁵ м = 0,12
    • Найдем θ: θ = arcsin(0.12) ≈ 6,89°
  • Ответ: ≈ 6,89°

Задача 3

На дифракционную решетку с периодом 2 мкм падает нормально монохроматический свет. Угол между первым и вторым максимумами равен 19,6°. Определите длину волны света.

  • Дано:
    • d = 2 мкм = 2 * 10⁻⁶ м (период решетки)
    • θ₂ - θ₁ = 19,6° (разность углов)
    • m₁ = 1 (первый максимум)
    • m₂ = 2 (второй максимум)
  • Найти: λ (длина волны)
  • Решение:
    • Для первого максимума: d * sin(θ₁) = m₁ * λ = λ
    • Для второго максимума: d * sin(θ₂) = m₂ * λ = 2λ
    • Так как θ₂ - θ₁ = 19.6°, то θ₂ = θ₁ + 19.6°
    • Выразим λ из первого уравнения и подставим во второе: d * sin(θ₂)= 2 * (d * sin(θ₁))
    • или sin(θ₂) = 2 * sin(θ₁)
    • Подставим θ₂ = θ₁ + 19.6°: sin(θ₁ + 19.6°) = 2 * sin(θ₁)
    • Решая уравнение (численно, так как аналитическое решение сложное), получим θ₁ ≈ 28.3°
    • Тогда λ = d * sin(θ₁) = 2 * 10⁻⁶ м * sin(28.3°) ≈ 9,44 * 10⁻⁷ м = 944 нм
  • Ответ: ≈ 944 нм

Задача 4

На щель падает монохроматический свет. Ширина щели равна 0,2 мм. Определите длину волны света, если первый дифракционный минимум наблюдается под углом 0,3°.

  • Дано:
    • a = 0,2 мм = 0,2 * 10⁻³ м (ширина щели)
    • θ = 0,3° (угол первого минимума)
    • m = 1 (первый минимум)
  • Найти: λ (длина волны)
  • Решение:
    • Условие минимума: a * sin(θ) = m * λ
    • Выразим λ: λ = (a * sin(θ)) / m
    • Вычислим: λ = (0,2 * 10⁻³ м * sin(0,3°)) / 1 ≈ 1,05 * 10⁻⁶ м = 1050 нм
  • Ответ: ≈ 1050 нм

Задача 5

Определите наибольший порядок дифракционного максимума для дифракционной решетки, содержащей 500 штрихов на 1 мм, если длина волны света равна 700 нм.

  • Дано:
    • N = 500 штрихов/мм = 5 * 10⁵ штрихов/м (плотность штрихов)
    • λ = 700 нм = 700 * 10⁻⁹ м (длина волны)
  • Найти: mmax (наибольший порядок максимума)
  • Решение:
    • Период решетки: d = 1 / N = 1 / (5 * 10⁵) м = 2 * 10⁻⁶ м
    • Условие максимума: d * sin(θ) = m * λ
    • Максимальный порядок соответствует sin(θ) = 1 (т.е., θ = 90°): d = mmax * λ
    • Выразим mmax: mmax = d / λ
    • Вычислим: mmax = (2 * 10⁻⁶ м) / (700 * 10⁻⁹ м) ≈ 2,86
    • Так как m должно быть целым, mmax = 2.
  • Ответ: 2

Задача 6

На дифракционную решетку нормально падает свет с длиной волны 500 нм. Под каким углом наблюдается максимум третьего порядка, если период решетки равен 1,5 мкм?

  • Дано:
    • λ = 500 нм = 500 * 10⁻⁹ м (длина волны)
    • d = 1,5 мкм = 1,5 * 10⁻⁶ м (период решетки)
    • m = 3 (порядок максимума)
  • Найти: θ (угол дифракции)
  • Решение:
    • Условие максимума: d * sin(θ) = m * λ
    • Выразим sin(θ): sin(θ) = (m * λ) / d
    • Вычислим: sin(θ) = (3 * 500 * 10⁻⁹ м) / (1,5 * 10⁻⁶ м) = 1
    • Найдем θ: θ = arcsin(1) = 90°
  • Ответ: 90°

Задача 7

Определите ширину щели, если при дифракции на ней свет с длиной волны 600 нм дает первый минимум под углом 0,25°.

  • Дано:
    • λ = 600 нм = 600 * 10⁻⁹ м
    • θ = 0,25°
    • m = 1
  • Найти: a
  • Решение:
    • a * sin(θ) = mλ
    • a = mλ / sin(θ)
    • a = (1 * 600 * 10⁻⁹ м) / sin(0.25°) = 0.000137 м = 0.137 мм
  • Ответ: ≈ 0.137 мм

Задача 8

На дифракционную решетку с периодом 3 мкм падает свет. Угол между первым и вторым максимумами равен 10°. Определите длину волны света.

  • Дано:
    • d = 3 мкм = 3 * 10⁻⁶ м
    • θ₂ - θ₁ = 10°
    • m₁ = 1
    • m₂ = 2
  • Найти: λ
  • Решение:
    • d sin θ₁ = m₁λ = λ
    • d sin θ₂ = m₂λ = 2λ
    • θ₂ = θ₁ + 10°
    • d sin (θ₁ + 10°) = 2λ = 2d sin θ₁
    • sin (θ₁ + 10°) = 2 sin θ₁
    • Решим численно для θ₁, получим θ₁ ≈ 28.5°
    • λ = d sin θ₁ = 3 * 10⁻⁶ м * sin (28.5°) = 1.43 * 10⁻⁶ м
  • Ответ: ≈ 1430 нм

Задача 9

Определите расстояние между штрихами дифракционной решетки, если при освещении ее светом с длиной волны 500 нм наблюдается дифракционный максимум первого порядка под углом 10°.

  • Дано:
    • λ = 500 нм = 500 * 10⁻⁹ м
    • θ = 10°
    • m = 1
  • Найти: d
  • Решение:
    • d * sin(θ) = m * λ
    • d = m * λ / sin(θ)
    • d = (1 * 500 * 10⁻⁹ м) / sin(10°) = 2.88 * 10⁻⁶ м = 2.88 мкм
  • Ответ: ≈ 2.88 мкм

Задача 10

Во сколько раз изменится расстояние между дифракционными полосами на экране, если дифракционную решетку заменить другой, имеющей в два раза больше штрихов на миллиметр?

  • Решение:
    • d = 1/N (где N — число штрихов на мм)
    • d₁ = 1/N₁
    • d₂ = 1/N₂
    • N₂ = 2N₁
    • d₂ = 1/(2N₁) = d₁/2
    • Расстояние между полосами обратно пропорционально периоду решетки.
    • Таким образом, расстояние уменьшится в 2 раза.
  • Ответ: В 2 раза уменьшится.
