Задачи на Дисперсию Света

Решите следующие задачи, чтобы закрепить свои знания о дисперсии света.

Задача 1

Белый свет падает на стеклянную призму. Красный свет испытывает угол отклонения 30°, а фиолетовый – 32°. Определите угловую дисперсию призмы.

  • Дано:
    • δкр = 30° (угол отклонения красного света)
    • δф = 32° (угол отклонения фиолетового света)
  • Найти: D (угловая дисперсия)
  • Решение:
    • Угловая дисперсия: D = δф - δкр
    • Вычислим: D = 32° - 30° = 2°
  • Ответ: 2°

Задача 2

Показатель преломления стекла для красного света равен 1.51, а для фиолетового – 1.53. Угол преломления призмы равен 5°. Определите угловую дисперсию призмы.

  • Дано:
    • nкр = 1.51 (показатель преломления красного света)
    • nф = 1.53 (показатель преломления фиолетового света)
    • A = 5° (угол призмы)
  • Найти: D (угловая дисперсия)
  • Решение:
    • Формула для угловой дисперсии при малых углах: D = A * (nф - nкр)
    • Вычислим: D = 5° * (1.53 - 1.51) = 0.1°
  • Ответ: 0.1°

Задача 3

При прохождении белого света через стеклянную призму, фиолетовый свет отклоняется сильнее, чем красный. Объясните это явление.

  • Следовательно, фиолетовый свет испытывает большее отклонение при прохождении через призму.
  • Ответ: Фиолетовый свет отклоняется сильнее из-за большей зависимости показателя преломления от длины волны.

    • Задача 4

    Угол призмы равен 10°. Показатель преломления для красного света 1.5, а для фиолетового 1.52. Найдите разность углов отклонения красного и фиолетового света.

    • Дано:
      • A = 10° (угол призмы)
      • nкр = 1.5
      • nф = 1.52
    • Найти: Δδ (разность углов отклонения)
    • Решение:
      • Формула отклонения: δ = A(n - 1)
      • δкр = A(nкр - 1) = 10°*(1.5 - 1) = 5°
      • δф = A(nф - 1) = 10°*(1.52 - 1) = 5.2°
      • Δδ = δф - δкр = 5.2° - 5° = 0.2°
    • Ответ: 0.2°

    Задача 5

    Объясните, почему при прохождении белого света через призму наблюдается радужная полоса.

    • Решение:
      • Белый свет состоит из света разных цветов, каждый из которых имеет свою длину волны.
      • При прохождении через призму происходит явление дисперсии, т.е. разложение белого света в спектр.
      • Показатель преломления призмы зависит от длины волны: чем меньше длина волны, тем больше показатель преломления.
      • Поэтому лучи разного цвета отклоняются на разные углы.
      • В результате на выходе из призмы мы наблюдаем радужную полосу, где фиолетовый свет отклоняется больше всего, а красный – меньше всего.
    • Ответ: Радужная полоса образуется из-за дисперсии света, то есть разложения белого света в спектр, когда разные цвета отклоняются на разные углы.

    Задача 6

    Какое из явлений иллюстрирует рисунок: интерференцию, дифракцию или дисперсию?

    • Решение:
      • На рисунке изображено разложение белого света на составляющие его цвета (красный, оранжевый, жёлтый, зелёный, голубой, синий, фиолетовый).
      • Это явление называется дисперсией света.
    • Ответ: Дисперсию.

    Задача 7

    Дисперсия света приводит к появлению радуги. Объясните, почему радугу можно наблюдать после дождя.

    • Решение:
      • Капли дождя действуют как маленькие призмы.
      • Когда солнечный свет попадает на капли дождя, происходит преломление и отражение света.
      • Из-за дисперсии света, солнечный свет разлагается в спектр, подобно тому, как это происходит в призме.
      • Разные цвета света испытывают разное отклонение, что приводит к образованию дуги (радуги), которую мы видим на небе.
    • Ответ: Радуга появляется после дождя, потому что капли воды в воздухе действуют как призмы, разлагая солнечный свет в спектр из-за дисперсии.

    Задача 8

    Угловая дисперсия призмы равна 3°. Если угол призмы равен 6°, определите разность показателей преломления для крайних лучей спектра.

    • Дано:
      • D = 3°
      • A = 6°
    • Найти: Δn
    • Решение:
      • D = A * (n₂ - n₁)
      • Δn = n₂ - n₁
      • Δn = D/A = 3°/6° = 0.5
    • Ответ: 0.5

    Задача 9

    На призму с углом 4° падает белый свет. Определите угловое расстояние между красным (n = 1.50) и фиолетовым (n = 1.52) лучами после прохождения через призму.

    • Дано:
      • A = 4°
      • nкр = 1.50
      • nф = 1.52
    • Найти: D (угловая дисперсия)
    • Решение:
      • D = A * (nф - nкр) = 4° * (1.52 - 1.50) = 0.08°
    • Ответ: 0.08°

    Задача 10

    Что произойдет с радужной полосой при прохождении белого света через призму, если призму поместить в воду? Укажите, как изменится угловое расстояние между красным и фиолетовым лучами.

    • между красным и фиолетовым лучами уменьшится.
  • Ответ: Угловое расстояние между красным и фиолетовым лучами уменьшится.
