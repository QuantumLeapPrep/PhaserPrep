Задачи на Интерференцию Света

Решите следующие задачи, чтобы закрепить свои знания об интерференции света.

Задача 1

В опыте Юнга расстояние между двумя когерентными источниками света равно 0,5 мм, а расстояние от источников до экрана – 2 м. Определите расстояние между интерференционными полосами, если длина волны света равна 600 нм.

  • Дано:
    • d = 0.5 мм = 0.5 * 10⁻³ м (расстояние между источниками)
    • L = 2 м (расстояние до экрана)
    • λ = 600 нм = 600 * 10⁻⁹ м (длина волны)
  • Найти: Δx (расстояние между полосами)
  • Решение:
    • Формула для расстояния между интерференционными полосами: Δx = (λ * L) / d
    • Вычислим: Δx = (600 * 10⁻⁹ м * 2 м) / (0.5 * 10⁻³ м) = 2.4 * 10⁻³ м = 2.4 мм
  • Ответ: 2.4 мм

Задача 2

На тонкую пленку падает свет с длиной волны 540 нм. Определите минимальную толщину пленки, при которой наблюдается интерференционный максимум в отраженном свете. Показатель преломления пленки равен 1,35.

  • Дано:
    • λ = 540 нм = 540 * 10⁻⁹ м
    • n = 1.35 (показатель преломления)
  • Найти: d (минимальная толщина)
  • Решение:
    • Условие максимума для тонкой пленки (с учетом потери полуволны при отражении): 2nd = (m + 1/2)λ, где m = 0, 1, 2, ...
    • Для минимальной толщины m = 0: 2nd = λ/2
    • Выразим толщину: d = λ / (4n)
    • Вычислим: d = (540 * 10⁻⁹ м) / (4 * 1.35) = 100 * 10⁻⁹ м = 100 нм
  • Ответ: 100 нм

Задача 3

Два когерентных источника света с длиной волны 480 нм расположены на расстоянии 0,2 мм друг от друга. На каком расстоянии от источников нужно расположить экран, чтобы расстояние между интерференционными полосами было 1,2 мм?

  • Дано:
    • λ = 480 нм = 480 * 10⁻⁹ м
    • d = 0.2 мм = 0.2 * 10⁻³ м
    • Δx = 1.2 мм = 1.2 * 10⁻³ м
  • Найти: L (расстояние до экрана)
  • Решение:
    • Формула для расстояния между интерференционными полосами: Δx = (λ * L) / d
    • Выразим расстояние до экрана: L = (Δx * d) / λ
    • Вычислим: L = (1.2 * 10⁻³ м * 0.2 * 10⁻³ м) / (480 * 10⁻⁹ м) = 0.5 м
  • Ответ: 0.5 м

Задача 4

Определите, на сколько изменится разность хода двух интерферирующих лучей, если один из них пройдет дополнительное расстояние в 1,5 мкм в среде с показателем преломления 1,4.

  • Дано:
    • ΔL = 1.5 мкм = 1.5 * 10⁻⁶ м (дополнительное расстояние)
    • n = 1.4 (показатель преломления)
  • Найти: Δd (изменение разности хода)
  • Решение:
    • Изменение разности хода: Δd = n * ΔL - ΔL = (n - 1) * ΔL
    • Вычислим: Δd = (1.4 - 1) * 1.5 * 10⁻⁶ м = 0.4 * 1.5 * 10⁻⁶ м = 0.6 * 10⁻⁶ м = 0.6 мкм
  • Ответ: 0.6 мкм

Задача 5

Свет с длиной волны 630 нм падает на мыльную пленку под углом, близким к нормальному. Какова минимальная толщина пленки, при которой она будет казаться черной (минимум в отраженном свете)? Показатель преломления мыльного раствора равен 1,33.

  • Дано:
    • λ = 630 нм = 630 * 10⁻⁹ м
    • n = 1.33
  • Найти: d (минимальная толщина)
  • Решение:
    • Условие минимума для тонкой пленки (с учетом потери полуволны при отражении): 2nd = mλ, где m = 1, 2, 3,...
    • Для минимальной толщины m = 1: 2nd = λ
    • Выразим толщину: d = λ / (2n)
    • Вычислим: d = (630 * 10⁻⁹ м) / (2 * 1.33) = 236.8 * 10⁻⁹ м ≈ 237 нм
  • Ответ: ≈ 237 нм

Задача 6

Разность хода двух интерферирующих лучей составляет 1,8 мкм. Определите, усилится или ослабнет свет в точке наблюдения, если длина волны равна 600 нм.

  • Дано:
    • Δd = 1.8 мкм = 1.8 * 10⁻⁶ м
    • λ = 600 нм = 600 * 10⁻⁹ м
  • Определить: Усилится или ослабнет
  • Решение:
    • Определим порядок интерференции: m = Δd / λ = (1.8 * 10⁻⁶ м) / (600 * 10⁻⁹ м) = 3
    • Так как m – целое число, наблюдается максимум интерференции (усиление).
  • Ответ: Свет усилится

Задача 7

В установке Юнга расстояние между щелями 1 мм, а расстояние от щелей до экрана 3 м. На экране наблюдается интерференционная картина. Если длина волны света составляет 500 нм, то на каком расстоянии от центрального максимума будет находиться первый максимум?

  • Дано:
    • d = 1 мм = 1 * 10⁻³ м
    • L = 3 м
    • λ = 500 нм = 500 * 10⁻⁹ м
    • m = 1 (первый максимум)
  • Найти: x (расстояние от центрального максимума)
  • Решение:
    • Расстояние от центрального максимума до m-го максимума: x = mλL/d
    • x = 1 * 500 * 10⁻⁹ м * 3 м / (1 * 10⁻³ м) = 1.5 * 10⁻³ м = 1.5 мм
  • Ответ: 1.5 мм

Задача 8

На тонкую пленку с показателем преломления 1,4 падает свет с длиной волны 560 нм. Определите минимальную толщину пленки, при которой она будет казаться наиболее яркой в отраженном свете (условие максимума). Угол падения света близок к нормальному.

  • Дано:
    • n = 1,4
    • λ = 560 нм = 560 * 10⁻⁹ м
  • Найти: d (минимальная толщина)
  • Решение:
    • Условие максимума в отраженном свете для тонкой пленки (с учетом смены фазы на π): 2nd = (m + 1/2)λ, m = 0, 1, 2...
    • Для минимальной толщины m = 0, тогда 2nd = λ/2
    • d = λ / (4n) = 560 * 10⁻⁹ м / (4 * 1,4) = 100 * 10⁻⁹ м = 100 нм
  • Ответ: 100 нм

Задача 9

Две стеклянные пластины приложены друг к другу под небольшим углом, образуя воздушный клин. На клин нормально падает монохроматический свет с длиной волны 580 нм. Расстояние между соседними интерференционными полосами равно 1,2 мм. Определите угол между пластинами.

  • Дано:
    • λ = 580 нм = 580 * 10⁻⁹ м
    • Δx = 1,2 мм = 1,2 * 10⁻³ м
  • Найти: α (угол между пластинами)
  • Решение:
    • Расстояние между полосами для воздушного клина: Δx = λ / (2α), где α – угол между пластинами.
    • α = λ / (2Δx) = 580 * 10⁻⁹ м / (2 * 1,2 * 10⁻³ м) = 2,42 * 10⁻⁴ рад
  • Ответ: 2,42 * 10⁻⁴ рад

Задача 10

Что произойдет с интерференционной картиной, если в опыте Юнга один из источников света заменить на источник с большей длиной волны?

  • Решение:
    • Расстояние между интерференционными полосами пропорционально длине волны: Δx = λL/d
    • Если длина волны увеличится, то и расстояние между полосами увеличится.
  • Ответ: Расстояние между интерференционными полосами увеличится.
