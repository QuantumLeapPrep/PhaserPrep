Задачи повышенной сложности (ОГЭ) по волновой оптике

Решите следующие задачи, чтобы проверить свои знания.

Задача 1: Интерференция

На экране наблюдается интерференционная картина от двух когерентных источников света. Расстояние между источниками 0.5 мм, расстояние до экрана 1 м. Определите длину волны, если расстояние между соседними интерференционными полосами (максимумами) равно 1 мм.

  • Дано:
    • d = 0.5 мм = 0.5 * 10⁻³ м
    • L = 1 м
    • x = 1 мм = 1 * 10⁻³ м
  • Найти: λ
  • Решение:
    • Формула для расстояния между полосами: x = λL / d
    • Выражаем длину волны: λ = xd / L
    • λ = (1 * 10⁻³ м * 0.5 * 10⁻³ м) / 1 м = 0.5 * 10⁻⁶ м
  • Ответ: 0.5 мкм

Задача 2: Дифракция на щели

На щель шириной 0.2 мм падает нормально монохроматический свет. Угол между направлениями на первый дифракционный минимум и максимум интенсивности равен 15°. Определите длину волны света.

  • Дано:
    • a = 0.2 мм = 0.2 * 10⁻³ м
    • φ = 15°
  • Найти: λ
  • Решение:
    • Условие минимума: a * sin(θ) = kλ, где k=1 для первого минимума
    • Условие максимума (приближенно): угол между первым максимумом и минимумом это половина угла между двумя первым минимумами, θ ≈ φ
    • Поэтому угол минимума примерно равен 2*15/2 = 15 градусов, значит sin(θ) = sin(15°)
    • λ = a * sin(θ)
    • λ = 0.2 * 10⁻³ м * sin(15°) ≈ 0.2 * 10⁻³ м * 0.2588 ≈ 5.18 * 10⁻⁵ м
  • Ответ: Примерно 51.8 мкм

Задача 3: Дифракционная решетка

На дифракционную решетку, имеющую 500 штрихов на миллиметр, падает нормально монохроматический свет с длиной волны 500 нм. Определите общее число дифракционных максимумов, которые можно наблюдать с помощью этой решетки.

  • Дано:
    • N = 500 штрихов/мм = 5 * 10⁵ штрихов/м (плотность штрихов)
    • λ = 500 нм = 500 * 10⁻⁹ м (длина волны)
  • Найти: Общее число максимумов
  • Решение:
    • Период решетки: d = 1 / N = 1 / (5 * 10⁵ м⁻¹) = 2 * 10⁻⁶ м
    • Условие максимума: d * sin(θ) = m * λ
    • Максимальный порядок дифракции: sin(θ) = 1
    • Максимальный порядок: m_max = d / λ = (2 * 10⁻⁶ м) / (500 * 10⁻⁹ м) = 4
    • Поскольку максимумы наблюдаются с обеих сторон от центрального максимума (m = 0), общее число максимумов: 2 * m_max + 1
    • Общее число максимумов: 2 * 4 + 1 = 9
  • Ответ: 9

Задача 4: Интерференция в тонких пленках

На тонкую пленку с показателем преломления 1.33 падает нормально белый свет. При какой минимальной толщине пленки в отраженном свете будет наиболее усилен зеленый свет с длиной волны 520 нм?

  • Дано:
    • n = 1.33 (показатель преломления пленки)
    • λ = 520 нм = 520 * 10⁻⁹ м (длина волны зеленого света)
  • Найти: d_min (минимальная толщина пленки)
  • Решение:
    • Интерференция в тонких пленках: учитывается сдвиг фазы на π при отражении от среды с бОльшим показателем преломления.
    • Условие усиления в отраженном свете: 2 * n * d + λ/2 = m * λ, где m - целое число. Минимальной толщине соответствует m = 1.
    • Выразим толщину пленки: 2 * n * d = λ/2 d = λ/ (4 * n)
    • d = (520 * 10⁻⁹ м)/ (4 * 1.33) ≈ 97.7 нм
  • Ответ: ≈ 97.7 нм

Задача 5: Просветление оптики

На линзу с показателем преломления 1.6 наносят тонкую пленку с показателем преломления 1.3. При какой минимальной толщине этой пленки отражение света с длиной волны 550 нм будет минимальным (просветление оптики)?

  • Дано:
    • n_л = 1.6 (показатель преломления линзы)
    • n_п = 1.3 (показатель преломления пленки)
    • λ = 550 нм = 550 * 10⁻⁹ м
  • Найти: d
  • Решение:
    • Для минимального отражения (деструктивная интерференция) с учетом изменения фазы на π при отражении от среды с бОльшим показателем преломления: 2n_п * d = (m + 0.5)λ, где m = 0 для минимальной толщины
    • Выражаем толщину пленки: d = (λ / 4n_п)
    • d = (550 * 10⁻⁹ м) / (4 * 1.3) ≈ 105.77 * 10⁻⁹ м
  • Ответ: Примерно 106 нм
